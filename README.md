# 🎬 Fullstack Movie Watchlist App

## אודות הפרויקט
אפליקציית Full-stack לניהול רשימת צפייה בסרטים. המערכת מאפשרת למשתמשים להוסיף סרטים חדשים, לצפות בכל הסרטים הקיימים ברשימה, לחפש סרטים לפי שם, ולמחוק סרטים קיימים.

הפרויקט נבנה באמצעות:
* **Frontend:** React, Vite, React Router DOM, CSS
* **Backend:** Node.js, Express, MongoDB, Mongoose
